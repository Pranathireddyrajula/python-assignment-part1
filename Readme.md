## Python Assignment - Part 1: Python Basics & Control Flow

Theme: Student Grade Tracker

This project is a simple Python-based Student Grade Tracker created as part of an assignment.

It performs basic data processing and analysis using core Python concepts like loops, conditionals, and string manipulation.

## Features

- Cleans and formats raw student data
- Validates student names
- Calculates total, average, highest, and lowest marks
- Assigns grades based on marks
- Allows adding new subjects using user input
- Generates a class performance report
- Performs basic string operations on a sample essay

## How to Run

1. Open the file `part1_grade_tracker.py`
2. Run the file using Python:
   python part1_grade_tracker.py
